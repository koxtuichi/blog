# 【第14話】60歳まで開かないタイムカプセル<br>〜iDeCoという名の拘束衣〜

「その武器の名は、**iDeCo（イデコ：個人型確定拠出年金）**だ」

「イデコ……。NISAとセットでよく聞きますね」

「NISAが『利益にかかる税金がゼロになる』制度だとしたら、
iDeCoは**『投資する元手にかかる所得税や住民税まで安くなる』**という、さらに強力な節税メリットがある」

マスターは説明した。
例えば、毎月2万円をiDeCoで積み立てると、年収にもよるが、年間数万円の税金が戻ってくる。
実質、**「投資するだけで、毎年お金がもらえる」**ようなものだ。

「ええっ！ そんなにお得なんですか！
じゃあNISAよりこっちを優先した方がいいんじゃ……」

「だが、最大のデメリットがある。
iDeCoに入れたお金は、**原則60歳になるまで絶対に引き出せない**」

「えっ……1円も？」

「1円もだ。
結婚しようが、家を建てようが、病気になろうが、おろせない。
だから私はこれを**『拘束衣』**と呼んでいる」

僕は息を飲んだ。
60歳。今の僕からすると、遥か彼方の未来だ。

「でもな、ケンタ君。
これはデメリットであると同時に、最強のメリットでもあるんだ」

「どういうことですか？」
お金が使えないことがメリット？

「人間は弱い生き物だ。
手元にお金があると、ついつい使ってしまう。
『車買っちゃおうかな』『リフォームしようかな』と理由をつけて、老後資金を取り崩してしまうんだ。
だが、iDeCoなら強制的にロックされる。
**『何があっても絶対に触れない老後のための聖域』**を作ることができるんだ」

「なるほど……。
最強の強制貯金システムってことですね」

「そうだ。
NISAは『いつでも使える自由な財布』。
iDeCoは『60歳まで開かないタイムカプセル』。
この2つを組み合わせることで、現在から老後まで、盤石な要塞を築くことができる」

「わかりました。
僕、iDeCoもやります。
老後の自分へのプレゼントだと思って」

「よし。
これで君は、攻め（株式）、守り（現金・債券）、節税（NISA・iDeCo）、出口戦略（サイドFIRE）。
すべての武器を手に入れたことになる」

マスターは窓の外を見上げた。
雨は完全に上がり、美しい夕焼けが広がっている。

「残るは、最後の一つ。
**『お金よりも大切なもの』**の話をして、この物語を終わりにしようか」
