# 【最終話】雨上がりの空<br>〜物語は続いていく〜

あれから、10年の月日が流れた。

雨上がりの夕暮れ。
『純喫茶ミリオネア』の看板は、今日も静かに灯っている。

カウンターの中に立っているのは、白髪が増えたマスター……ではなく、
エプロンをつけた僕（ケンタ）だ。

あの日、マスターは「世界一周の旅に出る」と言って、僕にこの店を託して旅立った。
今の僕は、週に3日だけこの店を開き、残りの時間は家族と過ごしたり、好きな執筆活動をして暮らしている。
資産は十分にある。働く必要はないが、この場所が好きだから続けている。

カランコロン。

ドアが開き、疲れ果てた様子の若いサラリーマンが入ってきた。
スーツはずぶ濡れで、この世の終わりのような顔をしている。

「いらっしゃい」

僕は彼に、温かいコーヒーを差し出した。

「はぁ……。また今月もカツカツか……。
一生このままなのかな……」

彼は独り言のように呟いた。
10年前の僕と、まったく同じ顔をしていた。

僕はグラスを拭く手を止めて、彼に微笑みかけた。

「どうしたんだい。この世の終わりみたいな顔をして」

彼は驚いたように顔を上げた。

「マスター……。正直に言っていいですか。
僕、真面目に働くのがバカらしくなってきました……」

僕はカウンターの下から、一枚の紙とペンを取り出した。

「なるほどな。
それは君が**『足し算の世界』**で生きているからだよ」

「足し算……？」

「ああ。
ここから少し長くなるが、私の昔話をしようか。
ある雨の日に、一人の若者が、魔法使いに出会った話を」

窓の外には、大きな虹がかかっていた。
僕の物語はハッピーエンドで終わったが、彼の物語はここから始まるのだ。

**知識は力なり。**
**昨日の自分より、今日の自分が少しでも豊かであれば、未来は恐れるに足らない。**

さあ、始めよう。
君だけの、希望の物語を。

**(完)**
