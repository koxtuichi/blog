# 【第9話】その日は突然やってくる<br>〜含み損という名の拷問〜

投資を始めてから3年が経った。
僕（ケンタ）の資産形成は順調だった。

毎月3万円の積立に加え、ボーナス月には増額もした。
世界経済は好調で、スマホの画面を見るたびに資産が増えていた。
「含み益（まだ確定していない利益）」はプラス30%を超え、僕は有頂天だった。

「なんだ、投資なんて簡単じゃないか。
銀行に預けてた頃の自分がバカみたいだ」

会社の同僚にも「まだNISAやってないの？」なんて上から目線で語ったりもした。
自分はもう、立派な投資家気取りだった。

だが。
**その日**は、何の前触れもなくやってきた。

月曜日の朝。
目覚まし代わりにスマホを見ると、ニュースアプリの通知が埋め尽くされていた。

**『米国市場、歴史的な大暴落。ブラックマンデー以来の下落幅』**
**『世界同時株安へ。終わりの始まりか？』**

血の気が引いた。以前マスターから聞いていた「暴落」という言葉が頭をよぎる。
震える指で証券アプリを開く。

「うそ……だろ……」

昨日まで「+150万円」と輝いていた含み益が、一晩で消し飛んでいた。
それどころか、元本を割り込み、**「-50万円」**という赤い数字が表示されている。

50万円。
僕の手取り給料の2ヶ月分以上が、寝ている間に消えたのだ。

「いやだ……返してくれ……」

仕事中も気が気じゃなかった。
トイレの個室に駆け込み、何度もスマホを見る。
見るたびに数字は減っていく。

「これ以上減ったらどうしよう」
「明日になったらゼロになるんじゃないか？」

夕方、僕は逃げるように会社を出て、雨の中『純喫茶ミリオネア』へ走った。

「マスター！ 助けてください！
僕のお金が……お金が溶けてなくなっちゃいます！」

店に飛び込むなり叫んだ僕を、マスターは静かに見つめていた。
いつもの優しい目ではない。
どこか冷徹な、プロの投資家の目をしていた。

「……来たか。
ようこそ、**『本当の投資の世界』**へ」
